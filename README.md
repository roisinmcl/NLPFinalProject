# NLPFinalProject

`data/` contains data files to be used in the project. They are seperated into `train/`, `test/`, and `dev/`. They are separated by speaker, and the political party of the speaker is 'P' in the name of the file: `([0-9]\_)+P\w\w.txt`, where 'R' represents the Republican party, and 'D' represents the Democratic party.
